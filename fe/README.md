## 开发
```bash
    # 克隆项目
    git clone https://github.com/wentao978/cloud-admin.git

    # 安装依赖
    npm install
    可以通过如下操作解决npm速度慢的问题
    npm install --registry=https://registry.npm.taobao.org

    # 本地开发 开启服务
    npm run dev
```
浏览器访问 http://localhost:8000

## 发布
```bash

    # 构建生成环境
    npm run build
```
